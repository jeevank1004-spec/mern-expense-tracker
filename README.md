# MERN Expense Tracker

A full-stack expense tracking application built with MongoDB, Express, React, and Node.js.

## Features
- User authentication
- Add daily expenses
- Monthly summary
- REST API backend

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## How to Run
1. Clone the repo
2. Install dependencies
3. Run backend and frontend
